Understanding SHAP values

SHAP (SHapley Additive exPlanations) is a game-theoretic approach that explains the output of any machine learning model. It connects optimal credit allocation with local explanations using classical Shapley values from game theory.

Interpretation Guide:
  • Mean Absolute SHAP Value: Indicates the overall importance/magnitude of a feature's effect on model predictions.
  • Direction badge: Shows if high feature values correlate with an increased threat/anomaly score (Increases Risk) or normal profile (Decreases Risk).
  • Correlation Coefficient: Measures the linear strength of the relationship between feature values and their respective SHAP values.
